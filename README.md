# is-yarn-global

Check if installed by yarn globally without any `fs` calls

## Install

```bash
$ npm install is-yarn-global
```

## Usage

Just require it in your package.

```javascript
const isYarnGlobal = require('is-yarn-global');

console.log(isYarnGlobal());
```
